# First Example — The Symbol is not the Number

## The idea

We start from the simplest possible example.

Take the symbol:

```text
10

Most humans read this as “ten”.

But this is not always true.


---

Change the base

The meaning of 10 depends on the base:

base 2  -> 10 = 2
base 3  -> 10 = 3
base 10 -> 10 = 10
base 16 -> 10 = 16

So the same symbol gives different values.


---

What this proves

This proves something very simple but very important:

the symbol is not the number.

The symbol depends on the representation.
The number does not.


---

Why this matters

Humans are trained to think in one main representation (base 10).

So we often forget that what we see is only one version of the object.

An AI does not need to have this limitation.

It could:

look at many representations at the same time

detect patterns across them

choose the representation where the structure is simpler

solve the problem there



---

The key insight

same symbol != same value

and also:

same number != same representation


---

Conclusion

This is a very small example.

But it shows the core idea of this project:

representation is not reality.

Once this is clear, a bigger question appears:

what could be discovered if we stop fixing one representation in advance?
